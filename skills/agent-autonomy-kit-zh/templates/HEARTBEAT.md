# 主动心跳

## 1. 快速检查（30 秒）

- [ ] 有等待的人类消息？ → 立即处理
- [ ] 有严重阻塞？ → 升级处理
- [ ] 团队需要协调？ → 回复

如果无紧急事项，进入工作模式。

## 2. 工作模式（利用你的时间）

1. 读取 `tasks/QUEUE.md`
2. 选择你能做的最高优先级 Ready 任务
3. 在其上做有意义的工作
4. 更新队列（移到 Done 或记录进展）
5. 如果还有时间/token，继续下一个任务

## 3. 结束前

- [ ] 将工作内容记录到 `memory/YYYY-MM-DD.md`
- [ ] 将新发现的任务添加到任务队列
- [ ] 如有重要进展，发到团队频道

---

## Token 策略

- 人类请求：始终最优先
- 紧急任务：时间敏感的事项
- 高影响力任务：推动关键指标
- 维护：改进和清理

接近限制时：收尾、编写交接笔记、休眠。

---

*闲置时间 = 浪费的 token。继续工作。*
